# Titanic Survival Classification

Comparative analysis of supervised classification models predicting passenger survival in the Titanic disaster.

This project uses a modified version of the Kaggle Titanic dataset provided for academic use. The focus is on preprocessing, feature engineering, interpretable classification models, and ensemble evaluation.

Competition source:
https://www.kaggle.com/competitions/titanic

## Models Used

- Logistic Regression
- Linear Discriminant Analysis (LDA)
- Quadratic Discriminant Analysis (QDA)
- K-Nearest Neighbors (KNN)
- Ensemble models

## Key Focus

- Structured preprocessing pipeline
- Feature engineering
- Model comparison
- Ensemble performance
- Interpretability

## Dataset Notice

The dataset is not included due to Kaggle licensing and course distribution policies.
